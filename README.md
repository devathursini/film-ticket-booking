# 🎥 CineVerse — Smart Movie Booking Platform

## 🚀 Introduction

CineVerse is a sleek and interactive **movie ticket booking web application** designed to simulate a real-world booking experience.
It enables users to explore movies, select seats, and proceed through a complete booking workflow with a modern UI.

---

## ✨ Core Functionalities

✔️ Movie browsing with categorized listings
✔️ Real-time search and filtering
✔️ Interactive seat selection interface
✔️ Dynamic pricing system
✔️ Multiple payment options (simulation)
✔️ Smooth navigation with SPA-like experience

---

## 🧩 Tech Implementation

* **Frontend:** HTML5, CSS3, JavaScript
* **Architecture:** Single Page Application (SPA) approach
* **State Handling:** JavaScript variables and DOM manipulation

## 💳 Payment Module

The payment system supports:

* UPI (QR + ID display)
* Card (UI simulation)
* Net Banking (UI simulation)

> ⚠️ Note: This is a simulated environment — no real transactions are processed.
